# HOLAMUNDO-SOIP
Sistema ALPR (Reconocimiento Automático de Placas) en tiempo real con enfoque Edge AI. Implementado en Python (YOLOv8 + EasyOCR) optimizado para Raspberry Pi 5. Incluye backend asíncrono en PHP/MySQL para cotejo de robos y un Dashboard web de monitoreo reactivo con alertas visuales. Diseñado para alta fidelidad bajo condiciones climáticas adversas.
